### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**用户端功能：**
1. 登录
2. 注册
3. 首页
4. 个人中心（修改密码、个人信息）
5. 公告信息管理
6. 药品信息管理
7. 药品取退记录管理

**员工端功能：**
1. 登录
2. 注册
3. 首页
4. 个人中心（修改密码、个人信息）
5. 公告信息管理
6. 药品信息管理
7. 出入库管理
8. 药品退取记录管理
9. 用户管理

**管理员功能：**
1. 登录
2. 个人中心
3. 管理员管理
4. 基础数据管理（药品单位管理、公告类型管理、药品类型管理）
5. 公告信息管理
6. 药品信息管理
7. 药品取退记录管理
8. 出入库管理（出入库管理、出入库详情管理）
9. 用户管理
10. 员工管理
